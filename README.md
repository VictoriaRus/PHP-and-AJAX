# PHP_and_AJAX

### List of technologies
- PHP;
- AJAX;
- CSS (Bootstrap);
- APACHE;

### Short description
This is a Rest application: registration and authorization with a personal user page.
- Server: PHP.
- Front: AJAX.
- Database: json file.
The application has form validation.
